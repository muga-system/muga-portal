# Cliente App

Panel admin + Portal clientes.

## Setup

```bash
pnpm install
pnpm dev
```

Puerto: `3001`

## Variables

```bash
cp .env.example .env.local
```

Requiere:
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `INTERNAL_ALLOWED_EMAILS`
