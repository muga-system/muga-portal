## [Unreleased]

## [1.0.0] - 2026-04-20

### Added
- **Portal cliente** - Portal para que clientes vean proyectos, entregables, archivos y comentarios
- **Panel admin** - Dashboard para gestión de leads, proyectos y clientes
- **Sistema de leads** - Captura desde sitio público, aprobación manual, onboarding de clientes
- **Analytics** - Mapa mundial de leads con drill-down por provincia Argentina
- **Auth** - Autenticación con email/password via Supabase Auth

### Changed
- Limpieza de documentación obsoleta (eliminado docs de wizard, figma, design)
- Actualización de URLs de CDN (GitHub → jsdelivr)

### Fixed
- Eliminación de código muerto (proxy.ts sin usar)
